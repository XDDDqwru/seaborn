# seaborn

### 1. Gráficos de Relación (Relational)
Se usan para ver cómo se relacionan las variables numéricas.
* **Scatterplot:** Puntos individuales.
* **Lineplot:** Ideal para tendencias temporales.

http://googleusercontent.com/image_collection/image_retrieval/9098440329281373899_0

### 2. Gráficos Categóricos (Categorical)
Para comparar grupos o categorías.
* **Barplot:** Barras con intervalos de confianza.
* **Boxplot:** Cajas que muestran cuartiles y valores atípicos.
* **Violinplot:** Muestra la densidad de los datos (la "forma" de la distribución).

http://googleusercontent.com/image_collection/image_retrieval/9098440329281373899_3

### 3. Gráficos de Distribución (Distribution)
Para ver cómo están repartidos los datos de una sola variable.
* **Histplot:** Histograma clásico.
* **Kdeplot:** Curva suave de densidad.

http://googleusercontent.com/image_collection/image_retrieval/9098440329281373899_4

### 4. Gráficos de Matrices (Regression & Correlation)
* **Heatmap:** Cuadrícula de colores para ver correlaciones.
* **Regplot:** Puntos con una línea de tendencia (regresión).

http://googleusercontent.com/image_collection/image_retrieval/9098440329281373899_6

### 5. Gráficos de Paneles Múltiples
* **Pairplot:** Compara todas las variables de un tirón.
* **Jointplot:** Mezcla dispersión y distribución en los bordes.

http://googleusercontent.com/image_collection/image_retrieval/9098440329281373899_1
